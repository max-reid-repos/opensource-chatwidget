Chat Widget Demo 💬

Stack-agnostic chat widget running on Express.js

Status

Backend running

The chat widget is in the bottom-right corner. Click it to start chatting!

How It Works

This demo includes:

  • Frontend: chat-widget.umd.js (20KB, zero dependencies)
  • Backend: Express.js with SQLite storage
  • Security: HMAC-signed tokens, rate limiting, fingerprinting

Integration Code

<script src="chat-widget.umd.js"></script>
<script>
  ChatWidget.init({
    apiUrl: '/api/chat',
    teamName: 'My Support',
    avatarInitials: 'MS',
    categories: [
      { id: 'sales', label: 'Sales', icon: '💰', description: 'Pricing' },
      { id: 'support', label: 'Support', icon: '🛠️', description: 'Get help' },
    ]
  });
</script>

Admin Endpoints

For testing, the backend exposes admin endpoints:

  • GET /api/admin/conversations — List all chats
  • POST /api/admin/reply — Send admin reply
curl http://localhost:3000/api/admin/conversations
